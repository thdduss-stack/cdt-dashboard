랜딩페이지 CTA 전환 퍼널

핵심 발견

PC의 상세→지원 전환율(~50%)이 MO(~35%)보다 압도적으로 높습니다. 그러나 MO의 리스트→상세 전환율(~43%)은 PC(~38%)보다 높아, 모바일 사용자는 공고를 더 많이 클릭하지만 지원 단계에서 이탈합니다. 이는 모바일 지원 CTA 버튼의 UX 개선이 필요하다는 강한 시그널입니다. 2025년 3월 대비 2026년 3월 MO total CVR이 14.99% → 10.01%로 하락 추세도 주의가 필요합니다.

Metabase SQL 쿼리 모음

아래 쿼리들을 Metabase에서 직접 실행하여 대시보드 카드로 저장할 수 있습니다.

1. 배너 CTR 분석

SELECT banner_no, banner_title, service_name, client_type, SUM(impression_count) as total_impressions, SUM(click_count) as total_clicks, ROUND(CAST(SUM(click_count) AS DOUBLE) / NULLIF(SUM(impression_count), 0) * 100, 2) as ctr_pct FROM box_dmp.ads_banner_smartfit_daily_summary WHERE impression_count > 0 GROUP BY banner_no, banner_title, service_name, client_type ORDER BY total_impressions DESC

2. 채용페이지 퍼널 CVR (월별)

SELECT DATE_TRUNC('month', event_date) as month, channel, gubun, SUM(list_pvs) as total_list_pvs, SUM(jobview_pvs) as total_jobview_pvs, SUM(applycomplete_pvs) as total_apply_complete, ROUND(CAST(SUM(jobview_pvs) AS DOUBLE) / NULLIF(SUM(list_pvs), 0) * 100, 2) as list_to_detail_cvr, ROUND(CAST(SUM(applycomplete_pvs) AS DOUBLE) / NULLIF(SUM(jobview_pvs), 0) * 100, 2) as detail_to_apply_cvr, ROUND(CAST(SUM(applycomplete_pvs) AS DOUBLE) / NULLIF(SUM(list_pvs), 0) * 100, 2) as total_cvr FROM self_insight_data.ga_jk_recruitment_page_applyfunnel WHERE event_date >= DATE '2025-01-01' GROUP BY DATE_TRUNC('month', event_date), channel, gubun ORDER BY month DESC, channel

3. 트래픽 소스별 방문

SELECT chnnel, user_type, COUNT(*) as visit_count, COUNT(DISTINCT user_pseudo_id) as unique_visitors FROM self_insight_data.ga_jk_visit_fact WHERE event_date >= '20250101' GROUP BY chnnel, user_type ORDER BY visit_count DESC

4. 공고추천 퍼널 (메일/푸시)

SELECT f_date, f_item, f_target, f_no, f_name, f_cnt FROM self_insight_data.funnel_snapshot WHERE f_date >= DATE '2025-01-01' ORDER BY f_date DESC, f_target, f_no

5. 노출 & 매출 (일별)

SELECT base_date, origin_site, SUM(impression_count) as total_impressions, SUM(total_sales_amount) as total_sales, ROUND(AVG(effective_cpm), 2) as avg_ecpm FROM box_performance_hub.exposure_stats WHERE base_date >= DATE '2025-01-01' AND _row_status = 1 GROUP BY base_date, origin_site ORDER BY base_date DESC

6. 유/무료/배너 클릭 일별

SELECT cl_date, pay_gi_cnt, pay_click_cnt, nopay_gi_cnt, nopay_click_cnt, banner_gi_cnt, banner_click_cnt, preview_cnt, totalsearch_cnt FROM box_job_statistics.stat_guin_click_cnt_new WHERE cl_date >= TIMESTAMP '2025-01-01' ORDER BY cl_date DESC

7. 채용페이지 UI 요소별 클릭

SELECT event_date, category, action, label, SUM(clicks) as total_clicks FROM self_insight_data.ga_jk_recruitment_page_clicks WHERE event_date >= DATE '2025-01-01' GROUP BY event_date, category, action, label ORDER BY total_clicks DESC
